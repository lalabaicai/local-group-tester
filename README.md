# 图的局部群成员判定器

一个交互式 Web 应用，用于可视化构建图、输入置换，并判定给定置换是否属于某个顶点的局部群。

## 项目概述

给定一个简单连通图 $G=(V,E)$，每条边 $\{u,v\}$ 对应对称群 $S(V)$ 中的对换 $(u\,v)$。顶点 $v$ 的**局部群** $G_v$ 定义为所有基于 $v$ 的闭途径的对换乘积之集合。

本应用实现了局部群成员判定的**完全分类定理**：
- **桥边块**：贡献平凡群
- **单环块 $C_n$**：贡献循环群 $\mathbb{Z}/(n-1)\mathbb{Z}$
- **多环二部块**：贡献交错群 $A(V_B)$
- **多环非二部块**：贡献对称群 $S(V_B)$

## 使用指南

### 1. 图编辑器

在左侧的画布区域进行图的可视化编辑：

- **创建节点**：点击画布空白处，会在该位置创建一个编号节点（编号从 1 开始自动递增）
- **选择节点**：点击一个节点将其设为**待判定点**（红色高亮）
- **连接节点**：再次点击已选中的节点进入**连接模式**（黄色高亮），然后点击另一个节点即可创建边。如果两节点之间已有边，再次连接不会重复创建
- **移动节点**：拖拽节点可改变其位置
- **删除节点**：右键点击节点可删除该节点及其关联的所有边。删除后编号会自动重新排列以保持连续性
- **清空画布**：点击右上角的"清空"按钮删除所有节点和边

### 2. 待判定点选择

在右侧面板的"待判定点 v"区域：

- 点击任意节点编号按钮选择该顶点作为待判定点
- 选中后会在下方显示该点局部群的结构描述

### 3. 待判定置换输入

在"待判定置换 π"区域：

- 输入框中填写 $\pi(i)$ 的值，即置换将 $i$ 映射到的值
- 每个输入框接受 1 到 $n$ 的整数，且所有值必须不重复
- **随机**：生成一个随机置换
- **恒等**：重置为恒等置换

### 4. 执行判定

点击"**判定 π 是否属于 G_v**"按钮，系统将：

1. 对图进行**块分解**（Tarjan 算法），找出包含待判定点的所有块
2. 对每个块，根据定理判断置换在该块上的限制是否满足条件
3. 显示判定结果和详细的数学推理过程

### 5. 结果解读

判定结果包含：

- **结论**：置换是否属于局部群的最终判定
- **各块条件验证**：对每个块的具体分析
  - 单环块：验证置换限制是否为生成 $(n-1)$-循环的某次幂
  - 二部多环块：验证置换限制是否为偶置换
  - 非二部多环块：自动满足（任何固定 $v$ 的置换限制）

## 技术说明

- **前端框架**：React + TypeScript + Tailwind CSS
- **图算法**：Tarjan 块分解、二部图检测、置换奇偶性计算
- **判定算法**：基于块分解的局部群结构定理实现
- **纯前端**：所有计算在浏览器中完成，无需后端服务

## 数学理论基础

局部群的完全分类定理：

$$G_v = \prod_{B \in \mathcal{B}(v)} \mathcal{L}(B,v)$$

其中 $\mathcal{B}(v)$ 是包含 $v$ 的所有块，$\mathcal{L}(B,v)$ 由各块类型唯一确定。

置换 $\pi \in G_v$ 当且仅当：
1. $\pi(v) = v$（固定 $v$）
2. 对每个块 $B$，$\pi|_{V(B) \setminus \{v\}} \in \mathcal{L}(B,v)$
